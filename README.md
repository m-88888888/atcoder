# Atcoder Practice

## Require

- [atcoder-cli](https://github.com/Tatamo/atcoder-cli)
- [online-judge-tools/oj](https://github.com/online-judge-tools/oj)

```sh
$ acc login
$ oj login https://atcoder.jp/
```

## Usage

### Get contest ID

```sh
$ acc new abc086
$ cd abc086

# write code...

# oj t -c "ruby main.rb" -d tests
$ acc s main.rb
```
